# what need does this project fill?

I want to know how i manage my money and I dont want to spend a lot of time doing so.
I think the best way to fix this is through the envelope method.

For those that dont know the envelope method is a money managing technique where you filter your money into literal envelopes. So you could have an envelope for groceries, eating out, and another for housing etc.
